# DynamicSmartFlow3 项目结构说明

## 目录角色

| 目录 | 职责 | 说明 |
|------|------|------|
| **config/** | 配置 | 按模块拆分（paths、chain、api_keys、trading、hunter 等），`settings.py` 统一导出 |
| **src/** | 外部集成 | RPC（Alchemy、Helius）、API（Birdeye、DexScreener）、Rugcheck 风控 |
| **services/** | 核心业务 | Monitor、Agent、Trader、猎手挖掘（modela/modelb）、通知 |
| **utils/** | 通用工具 | Logger、交易历史、Solana ATA 等，不含业务逻辑 |
| **tools/** | 脚本/CLI | 预留，健康检查等可放此处 |
| **data/** | 运行时数据 | modelA、modelB 子目录 |
| **DOCS/** | 文档 | Markdown 说明 |

## 猎手挖掘（MODELA / MODELB）——按模式分文件夹

```
services/
├── hunter_common/          # 共用：MODELA 与 MODELB 共享
│   ├── shared.py          # 交易解析、LP 检测、高频检测
│   └── __init__.py
├── modela/                # MODELA：高收益 token 挖掘
│   ├── searcher.py        # SmartMoneySearcher（DexScreener 热门币回溯）
│   ├── scoring.py        # MODELA 评分
│   └── __init__.py
├── modelb/                # MODELB：SM 榜单挖掘
│   ├── searcher.py        # SmartMoneySearcherB（wallets.txt 逐个分析）
│   ├── analyzer.py       # 三维度统计（盈利力/持久力/真实性）
│   ├── scoring.py        # MODELB 三维度评分
│   └── __init__.py
├── hunter_monitor.py
├── hunter_agent.py
├── trader.py
└── notification.py
```

- **共用**：`services/hunter_common/`（TransactionParser、LP、高频检测）
- **MODELA**：`services/modela/`（searcher + scoring）
- **MODELB**：`services/modelb/`（searcher + analyzer + scoring）

## 配置约定

- 所有配置通过 `from config.settings import X` 引入
- 不在业务代码中硬编码路径或常量
